This is a repository for the code used in the visualization of the geodesic flow in the unit tangent bundle of the hyperbolic upper-half plane. This code has been used in the final degree thesis:

Geodesic flows on hyperbolic Riemann surfaces

Author: Pau Arbonés Ocaña

Director: Gustavo Rodrigues Ferreira

Codirector: Núria Fagella Rabionet

Faculty of Mathematics of University of Barcelona
